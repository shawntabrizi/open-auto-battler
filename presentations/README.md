# Presentations

This folder contains markdown-based slide presentations for Open Auto Battler.

## Tools

These presentations are designed for lightweight markdown slide tools such as:

- [Marp](https://marp.app/) - Markdown Presentation Ecosystem
- [Slidev](https://sli.dev/) - Presentation Slides for Developers
- [reveal-md](https://github.com/webpro/reveal-md) - reveal.js with markdown

## Presentations

| Folder | Description |
|--------|-------------|
| `game-overview/` | Introduction to Open Auto Battler for team members |

## Creating a New Presentation

1. Create a new folder: `presentations/<presentation-name>/`
2. Add a `slides.md` file with your content
3. Add any assets (images, diagrams) to the same folder
4. Update this README with the new presentation
