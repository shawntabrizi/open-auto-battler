# Card Art Descriptions

Image prompt descriptions for generating card art. Each description focuses on the
character's visual identity — what you'd see in a portrait-style card frame.

**Style guide:** Each card should be a single character/creature centered in frame,
facing slightly toward the viewer. Background should be simple and themed to the faction.
Higher cost/rarity cards should look more imposing, detailed, and powerful.
